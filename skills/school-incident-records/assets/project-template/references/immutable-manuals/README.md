# immutable-manuals

Place school-approved reference materials here before using the skill.

Recommended materials:

1. Education activity protection or teacher protection manual
2. School violence or student guidance manual
3. Internal school reporting procedure
4. NEIS cumulative record guidance
5. Local report template or style sample

Treat this folder as read-only. Add a new file when a manual changes; do not overwrite old manuals unless the school explicitly directs it.

Do not commit real manuals to a public GitHub repository unless they are already public and redistribution is permitted.
